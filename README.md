# MiniPhotoshop

A simple image editing application built with WPF and C#.

## Features

- **File Operations**: Open and save images (JPG, PNG, BMP, GIF)
- **Navigation**: Undo/Redo, Zoom in/out, fit-to-screen
- **Transformations**: Crop, resize, rotate, flip
- **Adjustments**: Brightness, contrast, saturation controls
- **Drawing Tools**: Brush, eraser, fill, text, shapes (rectangle, circle, line)
- **Color Tools**: Color picker and palette
- **Selection**: Rectangle select, simple lasso tool

## Requirements

- .NET 8.0 or later
- Windows OS

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kaisabiyyi/miniphotoshop.git
   cd miniphotoshop
   ```

2. Build the project:
   ```bash
   dotnet build
   ```

3. Run the application:
   ```bash
   dotnet run
   ```

## Usage

1. Launch the application
2. Click "Pilih Gambar" to open an image file
3. Use the toolbar to apply various editing operations
4. Save your work using the save functionality
